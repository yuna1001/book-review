# ブックレビュー
書籍レビューサイト
https://book-review.ml/

# 概要
書籍のレビューサイトです。

# 機能一覧
- ユーザ機能
  - 新規登録
  - 編集
  - 検索
  - ソーシャルログイン
  - フォロー機能
- 書籍検索機能
  - APIでの書籍検索
  - 書籍データの保存
  - コメント投稿
  - 書籍データの検索
# 使用している技術
- 言語/フレームワーク
  - Python3.7.3
  - Django2.2.1
- 開発環境
  - Docker
  - Docker-Compose
    - Webアプリケーション(Django)・Webサーバ(Nginx)・Database(PostgreSQL)
- CI/CD
  - CircleCI
- 本番環境
  - AWS
    - EC2
    - Route53
    - S3・CloudFront
    - SES
